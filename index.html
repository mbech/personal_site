eGlobalReader Analytics Page

An interactive dashboard for language-learning ebook-readers to track their usage stats

I created a single page, responsive analytics website for a language-learing ebook startup. Upon logging in, the site requests reader data from the server and displays it after applying user-selected data filters (date range, data type, etc.).

Languages/Libraries/Frameworks used included:

  • HTML, CSS (SASS), and Javascript
  • Bootstrap (front-end framework)
  • Javascript libraries:
    • JQuery (general scripting library)
    • D3.js and NVD3 (vector graphics and graph/chart library)

See my blog for a more detailed post about the project (with screenshots).
